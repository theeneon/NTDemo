# Ninja Tactics — Phase 0 baseline

Status: Ready for product approval  
Version: 1.0  
Prepared: July 2026

This directory is the source of truth for Phase 0. The browser review portal at the repository root presents the same decisions as a responsive, clickable review surface.

## Deliverables

- [Scope lock](scope-lock.md): the vertical slice, exclusions, constraints, and change-control rule.
- [Screen flow](screen-flow.md): the first-run path and seven low-fidelity screen specifications.
- [Combat rules](combat-rules.md): deterministic combat behavior and ambiguous interaction decisions.
- [Acceptance checklist](acceptance-checklist.md): measurable player-loop, stability, save, responsive, accessibility, and hosting gates.
- [Asset inventory](asset-inventory.md): placeholder art direction, quantities, formats, and priority.

## Exit gate

Phase 1 may begin after the product owner accepts these five files as the demo baseline. New ideas that exceed the baseline move to a post-demo backlog unless they replace, rather than add to, an item of comparable size.
